🛍️ Customer Behavior Analysis 

An end-to-end data analytics project where I explored customer shopping behavior using SQL, Python, and Power BI — starting from raw data to building an interactive business dashboard.

This project focuses on understanding:

Who the customers are

How they purchase

What categories generate the most revenue

How subscription status impacts sales

📌 Project Objective

The goal of this project was to analyze customer purchasing patterns and build a business-ready dashboard that answers key stakeholder questions such as:

What is the total number of customers?

What is the average purchase amount?

Which product categories drive the most revenue?

How does subscription status affect customer distribution?

Which age group contributes the most revenue?

🛠️ Tech Stack Used
Tool	Purpose
SQL	Data cleaning, transformation, aggregations
Python (Pandas, Matplotlib/Seaborn)	Exploratory Data Analysis (EDA)
Power BI	Interactive dashboard & business visualization
🔄 End-to-End Workflow
1️⃣ Data Extraction & Cleaning (SQL)

Handled missing values

Used GROUP BY, JOIN, and aggregation functions

Generated revenue metrics

Identified duplicate or inconsistent entries

Created structured datasets for analysis

2️⃣ Exploratory Data Analysis (Python)

Performed customer segmentation by:

Age group

Category

Subscription status

Calculated:

Average purchase amount

Revenue by category

Revenue by age group

Visualized trends before dashboard development

This step helped me understand the data deeply before moving to Power BI.

Young and middle-aged customers contribute the highest revenue.

Average purchase amount suggests mid-range spending behavior.
